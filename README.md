# AI-Powered Productivity Assistant

## 📌 Project Overview
This project is an AI-powered assistant designed to automate workplace tasks such as email writing, meeting summaries, and task planning.

## 🎯 Problem Statement
Professionals spend too much time on repetitive tasks like writing emails and summarizing information.

## 💡 Solution
This assistant uses AI tools like ChatGPT to:
- Generate emails
- Summarize meeting notes
- Plan tasks

## 🛠 Tools Used
- ChatGPT
- Notion AI (optional)
- GitHub

## ✨ Features
- Smart Email Generator
- Meeting Notes Summarizer
- Task Planner

## ⚠️ Responsible AI
- Outputs must be reviewed by users
- AI may produce incorrect or biased responses

## 📊 Outcome
Improves productivity by saving time and automating repetitive tasks.
